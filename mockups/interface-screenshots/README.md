## Color Scheme

Current interface uses:
- **Primary Purple**: `#4c51bf` (headers, hero backgrounds)
- **Dark Navy**: `#1a202c` (footer, dark sections)
- **Teal/Cyan**: `#00d1b2` (CTAs, buttons)
- **White/Light Gray**: Background and cards

## Typography

- Headers: Sans-serif (system fonts)
- Body: Sans-serif
- Code: Monospace

## Components Observed

- **Navigation**: Top navigation bar with logo and hamburger menu
- **Cards**: Feature cards, extension cards, server cards, blog cards
- **Buttons**: Primary (teal), secondary styles
- **Forms**: Search bars with icon buttons
- **Filters**: Tag pills, dropdown selects, toggles
- **Stats Badges**: Circular icons with numbers
- **Footer**: Multi-column layout with links and social icons
- **Modals/Overlays**: Feedback widget (bottom-left)

## Notes for Redesign

- Current design uses Bulma CSS framework
- Responsive layout with mobile hamburger menu
- Heavy use of purple gradients in hero sections
- Card-based layouts throughout
- Consistent spacing and padding
- Icon usage throughout (mix of Font Awesome and custom)
