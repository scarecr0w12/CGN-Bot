## üìù Form Builder - Professional Applications Made Simple

Say goodbye to clunky Google Forms and complicated bots. Our **Form Builder** puts a powerful, native Discord form system right at your fingertips.

### What Makes It Special

**Visual Designer**
- Drag-and-drop field ordering
- Multiple field types (text, dropdown, multiple choice)
- Real-time preview as you build

**Automated Workflows**
- Submissions appear in designated channels
- One-click approve/reject buttons
- Automatic role assignment on approval
- Webhook integration for external tools

**Complete Response Management**
- View all submissions in one dashboard
- Filter by status (pending/approved/rejected)
- Add review notes for your team
- Track who reviewed what and when

### Common Use Cases

**Staff Applications**
Perfect for recruiting moderators, admins, or support staff. Create professional application forms with experience questions, availability checks, and scenario responses.

**Event Registration**
Collect sign-ups for tournaments, game nights, or community events. Automatically assign participant roles and send confirmation messages.

**Member Screening**
Implement verification processes for new members. Ask how they found your server, ensure they've read the rules, and gather initial information.

**Feedback Collection**
Gather structured feedback from your community about events, changes, or new features. Organize responses for easy analysis.

### Key Features

‚ú® **No Coding Required** - Visual builder makes it easy for anyone  
‚ú® **Discord Native** - Forms appear as Discord modals, no external links  
‚ú® **Response Dashboard** - Review and manage all submissions in one place  
‚ú® **Webhook Integration** - Connect to Zapier, Make, or custom endpoints  
‚ú® **Tier-Based Limits** - 2 forms free, 5 on Tier 1, unlimited on Tier 2

### How It Works

1. **Create** - Design your form in the dashboard
2. **Deploy** - Users submit via Discord commands
3. **Review** - Approve/reject with one click
4. **Automate** - Roles assigned, webhooks triggered

## üöÄ Getting Started

### Social Media Alerts

1. Navigate to **Dashboard ‚Üí Social Alerts**
2. Click **"Add Alert"**
3. Choose platform and enter account name
4. Select notification channel
5. Done! Alerts start immediately

**Requirements:**
- API keys for Twitch/YouTube (free)
- Manage Server permission

### Form Builder

1. Go to **Dashboard ‚Üí Form Builder**
2. Click **"Create New Form"**
3. Add fields with our visual editor
4. Configure channels and auto-roles
5. Publish and share with your community

**Note:** Discord slash command integration for form submission coming soon. Currently managed via dashboard.
