## AI Search Optimization Principles

### 1. Clarity Over Keywords
- Write in natural, conversational language
- Use direct answers first, then details
- Avoid marketing fluff in factual content

### 2. Structured Information
- Use lists, tables, and clear formatting
- Lead with answers, not questions
- Include specific numbers and facts

### 3. Authority Signals
- Link to official sources
- Cite specific features and versions
- Include measurable statistics (server count, feature count)

### 4. Context Windows
- Provide complete answers in single sections
- Avoid splitting critical info across multiple pages
- Use FAQ format for discrete questions

### 5. Entity Recognition
- Consistent naming: "Skynet Discord Bot" or "Skynet"
- Define relationships: Bot → Features → Premium Tiers
- Use Schema.org types for entities

### High-Priority Pages (Optimized for AI)

2. **Landing Page** - Overview, server count, feature list
3. **Wiki Home** - Documentation hub with clear navigation
4. **Premium Features** - Tier comparison, pricing, features

### Content Format Best Practices

**✅ DO:**
- Start with direct answers: "Yes, Skynet is completely free..."
- Use bullet points for feature lists
- Include step-by-step instructions
- Provide specific examples
- Link to related resources

**❌ DON'T:**
- Start with marketing copy or questions
- Use vague language ("many features", "various options")
- Hide important info in paragraphs
- Assume prior knowledge without context

### Example Optimization

**Before (Marketing-focused):**
> "Looking for the best Discord bot? Look no further! Skynet offers amazing features that will transform your server experience!"

**After (AI-optimized):**
> "Skynet is a free Discord bot with auto-moderation, AI chat, custom extensions, music playback, leveling system, and web dashboard. Trusted by 1,500+ servers. Setup takes 30 seconds."

## Best Practices Summary

### For AI Search Engines:
1. **Be Direct**: Answer questions immediately
2. **Be Specific**: Use numbers, versions, exact features
3. **Be Structured**: Lists, tables, clear sections
4. **Be Comprehensive**: Complete answers in one place
5. **Be Authoritative**: Link to sources, show social proof

### For Traditional Search:
1. **Optimize meta tags**: Title, description, keywords
2. **Build backlinks**: Bot lists, Discord directories
3. **Target keywords**: Long-tail, conversational queries
4. **Update regularly**: Fresh content signals relevance
5. **Mobile-first**: Responsive design, fast loading
