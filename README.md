## Quick Start

```bash
# Clone the repository
git clone https://github.com/scarecr0w12/CGN-Bot.git
cd CGN-Bot

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your Discord bot token and MongoDB connection

# Start the bot
node master.js
```
